Genre Search

route: GET "/api/genre"
return: JSON
format: /api/genre?list={list_of_genres}

Filter Parameters

By list
/api?list=action|romance

This parameter must be present to continue. The values are separated by a pipe ("|")character. It is case-insensitive.


JSON properties

genres

(STRING) The list of genres that have been queried.

titles

(LIST) JSON data of the titles.

  • title

    (STRING) title name of the book

  • page

    (STRING) interwiki main page title of the book.

  • lastreviseddate

    (STRING) The data and time of the last revision, in UTC+0.

  • lastrevisedid

    (NUMBER) the unique id of the last revison.

  • pageid

    (NUMBER) the unique id of the main page of the book

Example response using

/api/genre?list=shoujo|comedy
