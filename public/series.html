Light Novel Search

route: GET "/api"
return: JSON
format: /api?title={light_novel_title}[&series={regex_of_series_name}][&volume={regex_of_volume_name}][&volumeno={volume_number}]

Filter Parameters

By title
/api?title=Zero_no_Tsukaima

This parameter must be present to continue.

By series (Optional)
/api?title=Chrome_Shelled_Regios&series=chrome|regios

This parameter can be a regex of the series name. The example above uses the pipe character.

By volume (Optional)
/api?title=Chrome_Shelled_Regios&series=chrome&volume=15|log

This parameter can be a regex of the volume name, which includes the volume number. The example above uses the pipe character.

By volume number (Optional)
/api?title=Chrome_Shelled_Regios&series=chrome&volumeno=1

This parameter must be a number. Other characters will be removed.

JSON properties

title

(STRING) The title of the light novel

status

(STRING)The translation status of the light novel -- active, idle, halted, abandoned, warning.

Note: Unable to differentiate between completed and active, since the data is not available in the wiki text.

author

(STRING) The author of the light novel

illustrator

(STRING) The illustrator of the light novel

synopsis

(STRING) The synopsis of the light novel

Note: If synopsis heading unavailable, it will show all paragraphs with the title.

one_off

(BOOL) Is the light novel is one-off?

Note: A one-off novel is defined as not having any listed volumes.

sections

(LIST) The light novel series available. The data is provided as nested lists which contain json to preserve the order. Each series contains a list of volumes and each volume contains a list of chapters.

Each chapter contains the wiki link to the page.

  • title

    (STRING) title of the series

  • books

    (LIST) volumes available for the series

    • title

      (STRING) title of the volume

      cover

      (LINK) absolute link to the cover image of the volume

      chapters

      (LIST) chapters available for the volume, including external links

      • title

        (STRING) title of the chapter

        page

        (STRING) title of the page of the chapter in the wiki or link of external page.

        linktype

        (STRING) type of link of the chapter page. Two options are available: internal and external

        link

        (LINK) absolute link to the chapter

cover

(LINK) absolute link to the cover image of the novel

Example response using

/api?title=Hyouka&volumeno=1
