Type and Language Search

route: GET "/api/category"
return: JSON
format: /api/category?[type={type_of_title}][language={type_of_language}]

Filter Parameters

By type (Optional)
/api/category?type=LIGHT_NOVEL

Either parameter may be present. Current available types includes: LIGHT_NOVEL, TEASER, ORIGINAL_NOVEL. It is case insensitive.

By language (Optional)
/api/category?language=ENGLISH

Either parameter may be present. Language must be given in it's english name.english name. It is case insensitive.


JSON properties

There are different responses for different parameter combinations.

Type

(JSON) Shows the language available for that particular type.

  • type

    (STRING) type of title, e.g. Light_novel, teaser, Original_novel.

  • language

    (LIST) languages available for that type.

Example response using

/api/category?type=LIGHT_NOVEL

Language

(JSON) Shows the types available for that particular language.

  • language

    (STRING) Language being queried

  • language

    (LIST) types available for that language.

Example response using

/api/category?language=ENGLISH

Type&Language

(JSON) Shows the titles available that is of type and language

  • type

    (STRING) type of the titles

  • language

    (LIST) language of the titles

  • titles

    (LIST) JSON data of the titles

    • title

      (STRING) title name of the book

    • page

      (STRING) interwiki main page title of the book.

    • lastreviseddate

      (STRING) The data and time of the last revision, in UTC+0.

    • lastrevisedid

      (NUMBER) the unique id of the last revison.

    • pageid

      (NUMBER) the unique id of the main page of the book

Example response using

/api/category?type=LIGHT_NOVEL&language=ARABIC

Tags Search

route: GET "/api/category"
return: JSON
format: /api/category?[titles={title}][list={tag1|tag2}[&genres={genre1|genre2|...}][&type={book_type}&language={language}]]

Filter Parameters

By title (Optional)
/api/category?title=Zero_no_Tsukaima

This will return a list of categories that the title has been tagged with.


JSON properties

(LIST) List of tags available for that particular title

Example response using

/api/category?title=Zero_no_Tsukaima

By tags (Optional)
/api/category?list=MF Bunko J&genres=harem

This will return a list of JSON of books with the tags.

You can use this with type and language search. However, note that you must specify both. or it will fail unless the type is Original_novel.

You can also use this with genre search.


JSON properties

genres

(STRING) The list of categories that have been queried.

titles

(LIST) JSON data of the titles.

  • title

    (STRING) title name of the book

  • page

    (STRING) interwiki main page title of the book.

  • lastreviseddate

    (STRING) The data and time of the last revision, in UTC+0.

  • lastrevisedid

    (NUMBER) the unique id of the last revison.

  • pageid

    (NUMBER) the unique id of the main page of the book

Example response using

/api/category?list=MF Bunko J&genres=comedy|harem&type=Light Novel&language=english
